Well written Javascript

http://partch.anu.edu.au.virtual.anu.edu.au/~u6409914/resources/scripts/gallery.js

Notes: The script I wrote excels in the following critereon:

  • Portability : The script is portable in the sense that you can use it on any website that has a galleryPreview, image-title and preview tiles elements. Furthermore, it doesn't need any dependencies to run which is a massive positive for portability.
  • Performance : The script is lightweight to reduce high resource consumption from solutions that utilize external libraries such as JQuery to run.
  • Readability : The script has been designed to be very readable with clear variable names, modular functions instead a string of unorganised calls. Furthermore, each function is heavily commented with a description.
  • Extendability : The script is extendable as I have written my functins to do basics like switching an image by index. These foundational functions make it easier to extend the script to do more as I have already done with tile generation for example. Additionally I am using ES6 Javascript Syntax with anonymous functions, ${} data binding and good variable etiquette with let and const where context is not relevant which should be less likely to be deprecated in the future.
